# Hidden Lake Chat

## Dependencies

1. Go library [github.com/rivo/tview](https://github.com/rivo/tview "tview") 
2. Go library [github.com/gdamore/tcell/v2](https://github.com/gdamore/tcell "tview") 
3. Go library [github.com/number571/go-peer](https://github.com/number571/go-peer "go-peer") 
4. Go library [github.com/number571/hidden-lake](https://github.com/number571/hidden-lake "hidden-lake") 

## Build and run

```bash
$ git clone https://github.com/number571/hidden-lake
$ cd hidden-lake/projects/hl-chat
$ go run ./cmd/hl-chat -n "<network-key>" -f "<database-file>"
```
